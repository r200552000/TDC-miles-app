SOP 戰略操作指南
1
首頁計算怎麼用
  • 先輸入消費金額、付款情境與消費類別，再按「開始計算」。
  • 系統會依目前內建規則，列出最適合的卡片與預估哩程/積分。
  • 若要把結果存起來，請按卡片下方的「記帳」。
  • 若是退刷或負向修正，可使用金額欄下方的「退刷 / 修正上限額度」。
2
外幣模式與匯率
  • 切換為「🌎 外幣」後，可直接輸入國家名稱或幣別代碼，例如：日本 / JPY、英國 / GBP。
  • 點擊「自動帶入匯率」後，系統會抓取匯率並顯示 API 提供的資料更新時間。
  • 下方會同步顯示本金折台幣、1.5% 手續費與預估總扣款金額。
  • 若手動改過匯率，系統會顯示「已手動覆寫匯率」提示。
3
監控、明細與一鍵進倉
  • 「明細」會保存尚未結算的本期紀錄。
  • 「監控」可查看各卡本期累積與上限使用情況，必要時可手動編輯已消耗額度。
  • 按下「一鍵進倉」後，系統會把本期成果轉入倉庫。
  • 匯豐旅人積分與匯豐 Live+ 積分在系統內為分開資產池，會各自顯示對應夥伴與比例。
4
倉庫與 AI 戰略模式
  • 倉庫分為「資產管理」與「兌換試算」兩部分。
  • 貼上 Gemini API Key 後，可使用「全局資產巔峰推薦」與「指定航線達成方案」。
  • AI 推薦會優先根據內建卡別可兌換之航司與既有資產進行分析。
  • 結果下方提供長榮、華航、國泰、ANA 與 ANA 換星盟連結，方便立即查詢。
5
重要風險提醒
  • 本程式以環哩匯社員常用之內建卡別與航司規則為主。
  • 自訂卡片僅提供基礎國內 / 海外回饋設定，不含通路門檻、排除項目、活動加碼、回饋上限等複雜條件。
  • 使用者自由新增之航司，系統辨識力有限，AI 推估錯誤率較高。
  • 所有回饋、轉點、兌換與開票結果，仍請以銀行與航空公司官方公告為準。
